# 🚀 PPT 实用工具箱：快速上手指南

本工具箱集成了 AI 还原和 NotebookLM 转换两大核心功能，助你高效处理 PPT。

---

## 🛠️ 第一步：环境准备 (只需运行一次)

在开始使用前，请确保已安装必要的依赖项：

```powershell
# 1. 进入项目根目录
cd c:\Users\xxzx-admin\Desktop\实用工具

# 2. 安装 Python 依赖
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

> [!TIP]
> 如果你在使用 `ppt_edit_cli` 进行语义识别，请确保在根目录创建 `.env` 文件并填入你的 `GEMINI_API_KEY=你的KEY`。

---

## 🤖 工具 A：AI PPT 还原器 (`ppt_edit_cli`)

**适用场景**：将不可编辑的**PDF、图片截图**还原为带文字层、背景图且可编辑的 PPT。

### 1. 处理 PDF 文件 (推荐)
直接将 PDF 转换为 PPT，工具会自动处理分页和去底。
```powershell
.\.venv\Scripts\python.exe -m ppt_edit_cli.main -i "./assets/你的文件.pdf" -o "还原后的PPT.pptx"
```

### 2. 处理图片文件夹
如果你有一堆 PPT 截图，可以将它们放在一个文件夹中处理。
```powershell
.\.venv\Scripts\python.exe -m ppt_edit_cli.main -i "./input_images" -o "还原后的PPT.pptx"
```

**✨ 核心优势**：
- **智慧去底**：自动擦除原图文字，防止新叠文字产生“残影”。
- **水印移除**：针对 NotebookLM 水印进行自动消除。

---

## 📓 工具 B：NotebookLM 转换器 (`notebook_editor`)

**适用场景**：将 NotebookLM 正常导出的结构化 PPT 转换为**自由排版模式**，解除原有的固定框架限制。

```powershell
.\.venv\Scripts\python.exe notebook_editor\main.py "./assets/原始文件.pptx"
```

**输出**：将在同目录下生成 `原始文件_editable.pptx`。

## 📂 项目文件分布说明

- **`assets/`**: 存放你的原始文件和产出结果。
- **`ppt_edit_cli/`**: AI 还原工具核心逻辑。
- **`notebook_editor/`**: NotebookLM 转换工具逻辑。
- **`docs/`**: 详细技术手册。
- **`tests/`**: 开发者测试脚本。

## ❓ 常见问题
- **运行报错 `ModuleNotFoundError`？** 请确保执行了第一步的依赖安装。
- **文字识别不准？** 请检查网络连接（Gemini 需要联网）或更换更高质量的基础图片。
- **转换速度慢？** 首次运行会下载 OCR 模型，后续运行会变快；PDF 处理时间取决于页数。
